# 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/vaishnavijha006-hub/ai-data-automation.git
```

Install dependencies:

```bash
npm install
```

Run development:

```bash
npm run dev
```

Build project:

```bash
npm run build
```

Run production:

```bash
npm run start
```
